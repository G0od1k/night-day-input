# Night-Day Input

CSS library for stylizing html input.

## Preview

![Preview](./preview/img/toggle.png)

## Usage

```
<head>
    <link rel="stylesheet" href="https://unpkg.com/night-day-input" />
</head>
<body>
    <input type="checkbox" class="night-day-switch">
</body>
```

## Available Input Types

- [ ] Checkbox
- [x] Switch
- [ ] Radio
- [ ] Text
